# mistlur

A simple mp3 player for macOS.

![Cast](doc/cast.gif)

## How to start

Run

```
./mistlur 1.mp3 2.mp3 3.mp3
```

to queue files `1.mp3`, `2.mp3` and `3.mp3`.
